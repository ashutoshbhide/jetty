Jetty and JavaEE Web Profile

Jetty implements aspects of the JavaEE specification, primarily the Servlet Specification. In recent releases of the JavaEE platform, a Web Profile has been introduced recognizing that many developers need only a subset of the many technologies under the JavaEE umbrella.

Whilst Jetty itself does not ship all of the Web Profile technologies, the architecture of Jetty is such that you can plug in 3rd party implementations to produce a container customized to your exact needs.

Jetty-9.0.x is targetted at Servlet Specification 3.0, with some support for 3.1 features which are fully delivered in Jetty-9.1.x.

Here is the matrix of JSRs for JavaEE-6 Web Profile, and how they relate to Jetty:

JSR 318 Enterprise JavaBeans 3.1 No Yes, eg OpenEJB
JSR 317 Java Persistance 2.0 (JPA) No Yes, eg Hibernate
